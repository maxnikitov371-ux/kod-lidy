README для бета-тестеров

Проект: Квест "Код Лиды"

Быстрый запуск без сервера:
1. Открой папку проекта `kode_lidy_project`.
2. Дважды кликни `index.html`.
3. Сайт откроется в браузере в режиме `file://`.

Структура проекта уже подготовлена для офлайн-запуска:
- данные квеста встроены в `js/quest-data.js`;
- загрузка через `fetch` не используется.

Нужная версия Python (только если нужен локальный сервер):
- Python 3.12 (ветка 3.12.x)

Команды PowerShell для установки Python 3.12:
1. Открой PowerShell от имени администратора.
2. Проверь, установлен ли Python:
   python --version
3. Если Python не найден, установи:
   winget install -e --id Python.Python.3.12
4. Перезапусти PowerShell.
5. Проверь версию:
   python --version
   Ожидаемо: `Python 3.12.x`

Опционально: запуск через локальный сервер:
1. Открой PowerShell в папке `E:\kode_lidy_project`.
2. Выполни:
   python -m http.server 8000
3. Открой:
   http://localhost:8000/

Правило оформления текста в проекте:
- Не используем кавычки-елочки (U+00AB, U+00BB), используем только обычные двойные кавычки `"`.
- Не используем длинные тире (U+2014, U+2013, U+2011), используем только короткое тире `-`.

Проверка перед коммитом:
1. Открой PowerShell в папке проекта.
2. Выполни:
   `pwsh -File .\scripts\text-style-check.ps1`
